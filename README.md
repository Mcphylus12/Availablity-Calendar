## Overview
Available at https://mcphylus12.github.io/Availablity-Calendar/

I zero backend tool for finding free dates in a group. 
- A leader loads the page and then creates a range of months.
- They can then share the url to members who fill in days they **Arent** available
- Once filled it in the url will update and can be imported into each others calendars to build a complete calendar representing when everyone is busy
